# Acceleration Signal Analysis

## Overview

This project analyzes acceleration data collected from a smartphone and detects walking steps using MATLAB.

## Structure

* data/: raw acceleration data
* src/: MATLAB code

## Method

* Time domain analysis
* FFT (frequency analysis)
* Butterworth low-pass filter
* Peak detection

## Result
  Time Domain Signal
  Frequency Domain (FFT)
  Step Detection
    * Steps: 161
    * Distance: 104 m

## How to run

1. Open MATLAB
2. Set Current Folder to project directory
3. Run `main.m`
